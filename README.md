# Calculator App
Modern calculator application designed for students with a clean and responsive user interface.

## Preview
A simple scientific calculator built using pure front-end technologies.  
The project focuses on user-friendly design, smooth interaction, and responsive layout.

## Features
- Basic arithmetic operations
- Square calculation (x²)
- Square root calculation
- Percentage calculation
- Positive / Negative toggle
- Responsive modern UI
- Clean gradient design

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project was created to practice front-end development skills and improve UI design thinking for real-world web applications.

## Future Improvements
- Add keyboard support
- Add calculation history
- Dark mode
- Scientific calculator functions
